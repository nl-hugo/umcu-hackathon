How blues and greens color the number of admissions.

A visual exploration of patients' living environment.

The data
LeeftijdCategorie: patient ages
opname_duur: number of admission days
Blue/green relative 100..500: the sq meters of green/blue in the patient's environment

The steps
filter data <> "-1"
compute values: color value * number of admission days
group number of admissions by age, color and distance

The charts

Color intensity: the darker the color, the more admission days

The (provisional) take-away
In general: the farther away the green and blue, the more admission days for you.
Blue seems to be a better predictor than green.
But, no obvious differences between age categories.

Todo
Divide by the number of patients per category for normalization.
Add more dimensions (gender, diagnosis, ...)
